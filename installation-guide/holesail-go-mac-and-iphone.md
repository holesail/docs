# üçé Holesail Go (Mac & iPhone)

Holesail Go is a client for connecting to holesail servers. It supports scanning QR code and is available on the [**Apple store**](https://apps.apple.com/us/app/holesail-go/id6503728841) and [**Holesail Downloads Page**](https://holesail.io/download/).

Currently, we support only Apple silicon but soon support for Apple Intel will be available as well.
